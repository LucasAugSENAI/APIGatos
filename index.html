Cadastrar Imagens

Na seção de cadastro, as imagens serão fornecidas de forma aleatória através de uma API integrada. Os usuários terão a opção de visualizar uma nova imagem aleatória ou salvar a imagem no banco. Ao encontrar uma imagem que desejam salvar, poderão clicar em "Salvar" e adicionar um título para a imagem. Estas imagens serão então armazenadas no banco de dados local.

Acessar o Cadastro

Galeria

A galeria exibirá as imagens que foram salvas pelos usuários. Será organizada em um layout de grade, permitindo aos usuários navegar visualmente por suas coleções.
